# NATO Phonetic Converter

A Java-based NATO Phonetic Encryption & Decryption System that converts normal text into NATO phonetic alphabet format and vice versa using file handling, HashMaps, reverse mapping, and custom exception handling.

---

# Features

- Encrypt normal text into NATO phonetic format
- Decrypt NATO phonetic text back to normal text
- File handling using `input.txt` and `output.txt`
- Custom exception handling
- Multi-line file support
- Number support (`0-9`)
- Punctuation support (`. , ! ?`)
- Case-insensitive decryption
- Space preservation using `/`
- Modular and scalable code structure
